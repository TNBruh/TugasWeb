### **Package**
* Pymongo untuk MongoDB
* Tkinter

### **How to run**
Jalankan mongod.exe kemudian jalankan Control.py

### **Feature**
* Mendaftarkan diri
* Membentuk tim
* Menambah dan mengurangi anggota
* Menghapus tim
* Membuat turnamen dengan menantang tim lain
* Membatalkan turnamen
